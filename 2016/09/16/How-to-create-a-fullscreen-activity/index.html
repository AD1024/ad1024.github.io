How to create a fullscreen activity

I added a splash screen to ShiYiQuan-Event today, and I wanted to make it be a full screen activity. However, lots of tutorials on the Internet no longer available. So I tried several times, and finally, succeeded.

The way to make an activity be a fullscreen activity is:

First step

Modify AndroidManifest.xml. Add ‘Theme’ attribute to the target activity to disable ActionBar in that activity.

<activity
      android:name=".Activities.SplashScreen"
      android:label="十一圈Event"
      android:screenOrientation="portrait"
      android:theme="@style/AppTheme.NoActionBar">
      <intent-filter>
          <action android:name="android.intent.action.MAIN" />
          <category android:name="android.intent.category.LAUNCHER" />
      </intent-filter>
</activity>

Second step

Add following code to hide system status bar. Note that you need to put this code before setting content of the activity.

this.getWindow().setFlags(WindowManager.LayoutParams.FLAG_FULLSCREEN, WindowManager.LayoutParams.FLAG_FULLSCREEN);
